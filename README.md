# Projeto – Previsão de Mercado Financeiro

## Executar

docker compose up --build

## Acessos

Dashboard:
http://localhost:8501

API:
http://localhost:8000/docs

## Funcionalidades

- Seleção de ativo
- Seleção de modelo
- Filtro temporal
- Corte temporal t0
- Horizonte de previsão
- Visualização da previsão
- Intervalo temporal configurável
